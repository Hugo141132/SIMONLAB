About Us

Website ini dibuat untuk menyelesaikan project akhir kelas D4 Teknik Telekomunikasi A Angkatan 2023

Mata Kuliah Praktikum Komunikasi Data

Divisi Perancangan

Menyusun flow sistem secara menyeluruh, analisis kebutuhan perangkat keras dan lunak, serta mendefinisikan skenario penggunaan sistem.

Divisi Hardware

Merancang dan merakit sistem LED Running Text, timer, mikrokontroler (ESP32), serta konektivitas internet dan daya.

Divisi Embedded Programming

Pemrograman mikrokontroler untuk mengontrol LED display, membaca input jadwal, dan mengirim data ke server/website.

Divisi Web Development

Membangun antarmuka web untuk monitoring jadwal dan histori penggunaan lab, termasuk dashboard admin/asisten lab.

Divisi Integrasi & IoT

Menghubungkan sistem embedded dengan web server (API), sinkronisasi data cloud, serta pengujian komunikasi dua arah (ESP32-Web).

Divisi Telegram Bot

Membuat dan mengintegrasikan bot Telegram untuk mengirim notifikasi real-time kepada asisten lab, serta pengaturan akses.

Divisi Dokumentasi

Mengelola timeline proyek, dokumentasi teknis dan non-teknis (Keuangan), laporan perkembangan, serta komunikasi antar divisi.

Project ini diselesaikan oleh seluruh anggota kelas yang terbagi menjadi beberapa divisi seperti diatas.
